Web Skillup

Welcome to the Web Skillup program, presented by the Hyland Tech Outreach team!

Program Overview

This program is designed to cover essential web development concepts in HTML, CSS, and JavaScript. By the end of day two, all participants should have a solid grasp of the code in this project.

Given the short time frame, this will be a crash course. The hope is that each participant walks away feeling inspired and empowered to dive deeper into these topics and continue building their web development skills.

Each lesson is equipped with a 🟢 Basic and an 🔷 Advanced approach. Any participant, regardless of current knowledge, should be able to find value in this experience by taking one of these approaches.

Lessons

The program consists of the following lessons:

  • Lesson 1: HTML
  • Lesson 2: CSS
  • Lesson 3: Hello JavaScript!
  • Lesson 4: Variables & Data
  • Lesson 5: Buttons & Functions
  • Lesson 6: Conditionals
  • Lesson 7: DOM Manipulation
  • Lesson 8: For Loops
  • Lesson 9: Arrays
  • Lesson 10: Objects

Additional Challenges

Each lesson contains several opportunities to practice the concepts. Beyond that, there are also some additional challenges.

Additional Topics

In the event that all lessons have been completed before the end of the program, there are some additional topics to explore.
